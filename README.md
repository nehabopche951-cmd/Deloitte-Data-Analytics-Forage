# Deloitte Data Analytics Job Simulation (Forage)

## 📌 Project Overview
This project is based on the Deloitte Australia Data Analytics Job Simulation on Forage. The goal was to analyze employee compensation data and identify equality trends.

## 🛠 Tools Used
- Microsoft Excel  
- Tableau  

## 📊 What I Did
- Classified equality scores into Fair, Unfair, and Highly Discriminative using Excel  
- Created an interactive Tableau dashboard  
- Analyzed data to find inequality patterns  

## 🔍 Key Insights
- Identified areas with compensation inequality  
- Highlighted job roles with high discrimination scores  
- Provided data-driven business insights
